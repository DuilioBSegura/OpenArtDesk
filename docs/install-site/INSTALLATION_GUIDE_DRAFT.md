# Rascunho do Guia de Instalacao

Este documento e um rascunho. Ainda nao ha release publica, instalador, build .exe ou comandos finais de instalacao.

O site Como Instalar vira apos o primeiro build instalavel.

## Windows .exe futuro

Quando houver release:

1. Baixar o instalador .exe pela pagina oficial de releases.
2. Executar o instalador.
3. Abrir OpenArtDesk pelo atalho criado.
4. Confirmar onde os dados locais serao armazenados.

Status: futuro/rascunho.

## Linux futuro

Quando houver build Linux:

1. Baixar o pacote indicado para a distribuicao suportada.
2. Instalar conforme o formato publicado.
3. Abrir OpenArtDesk pelo menu do sistema.

Status: futuro/rascunho.
