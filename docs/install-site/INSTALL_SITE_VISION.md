# Visao do Site Como Instalar

## Objetivo

O site Como Instalar devera ajudar usuarios a baixar, instalar, entender e contribuir com OpenArtDesk depois que existir o primeiro build instalavel.

## Quando sera feito

O site sera criado apos o primeiro build instalavel Windows .exe estar disponivel e validado.

## Estilo visual

O site pode ter estilo de landing page/documentacao, com visual moderno e claro. Bootstrap pode ser usado como referencia visual ou base do site.

Bootstrap nao deve ser usado no app desktop.

## Secoes planejadas

- Home
- Como Instalar
- Windows
- Linux
- macOS futuro
- Como Usar
- Como Contribuir
- Politica de Conteudo
- Dados Locais
- Releases
- FAQ

## Stack futura possivel

- Astro
- VitePress

## Limites

O site nao deve prometer instalacao ou suporte para plataforma sem build real validado.
