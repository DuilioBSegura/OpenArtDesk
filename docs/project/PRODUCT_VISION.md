# Visao do Produto

## Nome

OpenArtDesk

## Tipo de produto

Aplicativo desktop open source, local-first e offline-first para organizacao de estudos, arquivos e materiais locais.

## Visao

OpenArtDesk ajuda estudantes a organizar conhecimento, arquivos e atividades de estudo no proprio computador, com uma experiencia visual, modular e customizavel.

## Promessa principal

Um desktop app open source para organizar conhecimento, arquivos e atividades de estudo localmente.

## Publico inicial

Estudantes de arte e estudos criativos que lidam com PDFs, referencias, atividades, arquivos locais, brushes, cursos e projetos pessoais.

## Problemas resolvidos

- Materiais espalhados em pastas locais.
- PDFs e referencias sem contexto.
- Dificuldade para acompanhar estudos e atividades.
- Falta de ferramenta desktop simples, privada e customizavel.
- Dependencia de ferramentas online para organizacao pessoal.

## Filosofia

- Local-first
- Offline-first
- Open source
- Customizavel
- Modular
- Visual
- Produtivo
- Privacidade por padrao
- Liberdade dos dados

## Identidade

OpenArtDesk deve parecer um aplicativo desktop limpo, moderno e produtivo, inspirado por apps Linux modernos, Obsidian, Zotero, Notion e Pomotroid, sem virar uma landing page ou web app disfarçada.

## Modulos do MVP

- Onboarding
- Dashboard
- Minha Biblioteca
- Estudos
- Referencias
- Atividades
- Customizacao
- Dados Locais e Privacidade
- Como Usar
- Configuracoes
- Backup completo

## Modulos futuros

- Meus Desenhos
- Meus Brushes
- Meus Cursos
- Meus Projetos
- Minhas Notas
- Minhas Aulas
- Minhas Colecoes
- Temas avancados
- Plugins em futuro distante

## Riscos principais

- Escopo crescer antes do MVP.
- Customizacao virar complexidade sem valor.
- Acoplamento entre UI, storage e filesystem.
- Prometer sync, IA ou plugins cedo demais.
- Tratar o app como web app em vez de desktop.
- Aceitar conteudo com risco de direitos autorais.

## Fora do MVP

- Login obrigatorio.
- Sync em nuvem.
- IA.
- Plugins.
- Leitor PDF embutido.
- Marketplace.
- Telemetria escondida.
- Backend remoto obrigatorio.

## MVP pronto

O MVP estara pronto quando o usuario puder usar um app desktop instalavel para organizar dados locais de estudo, biblioteca, referencias e atividades, com backup completo, privacidade clara e fluxo basico de customizacao.

## Frase-guia

Organizar estudos e materiais locais com liberdade, privacidade e uma mesa de trabalho que o usuario pode chamar de sua.
