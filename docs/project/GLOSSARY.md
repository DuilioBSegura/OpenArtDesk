# Glossario

OpenArtDesk: aplicativo desktop open source para organizar estudos, arquivos e atividades localmente.

Local-first: principio em que os dados principais vivem no dispositivo do usuario.

Offline-first: principio em que o app continua funcionando sem internet.

Material: item de estudo ou referencia, como PDF, imagem, link ou arquivo local.

Biblioteca: modulo para organizar PDFs e materiais locais.

Estudo: registro ou agrupamento relacionado a um objetivo de aprendizado.

Atividade: tarefa ou pratica associada aos estudos.

Referencia: material usado como inspiracao, consulta ou apoio.

Preferencias: escolhas de interface e experiencia do usuario.

Dados de dominio: registros estruturados do app, como materiais, estudos, tags e atividades.

Filesystem local: sistema de arquivos do computador do usuario.

Backup completo: exportacao de dados estruturados, preferencias e arquivos internos.

Module Registry: registro central de modulos, abas, rotas, labels e icones.

Customizacao: capacidade de adaptar visual, layout e comportamento da interface.

Tema: conjunto de cores, tokens e estilos visuais.

Layout: organizacao visual de areas, paineis e modulos.

Conteudo aberto: material com licenca aberta, dominio publico ou permissao clara de uso.

PDF local: arquivo PDF armazenado no computador do usuario ou copiado para pasta interna do app.
